HoloSplat — Embed Demo

JS API — player()

<script src="holosplat.iife.js"></script>

<div id="scene" style="width:100%; height:480px"></div>
<script>
  const p = HoloSplat.player('#scene', {
    src: 'https://your-cdn.com/scene.spz',
  });
</script>

Data-attribute auto-init (no JS required)

<script src="holosplat.iife.js"></script>

<div data-holosplat="https://your-cdn.com/scene.spz"
     style="width:100%; height:480px"></div>

The div below uses data-holosplat — same URL as above, loaded automatically.
